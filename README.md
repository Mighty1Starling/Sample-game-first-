# Игра «Лови особых рыбок»

Небольшая аркадная игра на Python с использованием библиотеки `pygame`.  
По экрану плывут разноцветные рыбки, задача игрока — за 60 секунд как можно больше раз кликнуть по **особым зелёным рыбам**, которые отличаются цветом и размером.

## Используемые технологии

- **Python 3.x** – основной язык разработки.  
- **Pygame** – библиотека для 2D‑графики, обработки ввода с клавиатуры и мыши, управления временем и игровым циклом.  
- Простейшее **хранение данных в JSON** для записи и чтения рекорда игрока (файл `fish_record.json`).

## Установка и запуск

1. Установить Python 3.x (если ещё не установлен).  
2. Установить зависимости:
   ```bash
   pip install pygame
   ```
3. Скопировать файл игры, например `fish_game.py`, и положить рядом с ним этот README.  
4. Запустить игру:
   ```bash
   python fish_game.py
   ```
5. После первого запуска в папке проекта автоматически появится файл `fish_record.json` с сохранённым рекордом.

## Правила игры

- На экране постоянно появляются и плывут слева направо рыбки.  
- Есть два типа рыб:
  - **Обычные** – красные, меньшего размера, клики по ним не учитываются.
  - **Особые** – зелёные и крупнее, дают очки за клик.
- Игрок кликает мышкой по особым рыбам и набирает очки.  
- В верхней части экрана отображаются:
  - текущий счёт (количество набранных очков);
  - оставшееся время (обратный отсчёт с 60 секунд);
  - личный рекорд (лучший счёт за все сессии, берётся из `fish_record.json`).  
- Когда время заканчивается:
  - игра останавливается;
  - на экране показываются финальный результат и сообщение о новом рекорде (если он побит);
  - можно перезапустить игру или выйти.

## Управление

- **ЛКМ (левая кнопка мыши)** – клик по рыбам (засчитываются только зелёные).  
- **R** – перезапуск игры после окончания раунда.  
- **Esc** – выход из игры (в любое время или после окончания).  
- Закрытие окна – также завершает игру.

## Основная структура кода

- `Fish` – класс, описывающий одну рыбу:
  - положение, скорость, тип (обычная или особая);
  - метод `update()` для движения;
  - метод `draw()` для отрисовки рыбы в стиле заданного образца (овал‑тело, зелёные треугольники‑плавники и хвост, глаз с жёлтым кольцом).  
- Игровой цикл:
  - обработка событий (клики мыши, выход, перезапуск);
  - таймер обратного отсчёта;
  - случайный спавн рыб;
  - обновление координат и удаление ушедших за экран рыб;
  - начисление очков при клике по особой рыбе.  
- Работа с рекордом:
  - `load_record()` – чтение рекорда из `fish_record.json` при запуске;
  - `save_record()` – сохранение нового рекорда по окончании раунда.

## Возможные улучшения

- Добавить звуковые эффекты при клике по особой рыбе и при окончании игры.  
- Ввести разные уровни сложности: скорость рыб, количество особых рыб, время раунда.  
- Добавить стартовое меню и экран настроек.  
- Нарисовать фон «под воду», а также разные варианты рыб (по цветам, форме и скорости).
