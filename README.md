# Fish Game - Простая казуальная игра-кликер на Python и Pygame

**Fish Game** — это простая 2D-игра на Python и Pygame для тренировки реакции и внимания. За 60 секунд игроку нужно набить максимальное количество очков, кликая по особым (зелёным) рыбам, которые плывут по экрану слева направо среди потока обычных красных рыб.[1]

Проект создан как **учебный пример** для начинающих разработчиков, демонстрирующий ключевые элементы разработки 2D-игр:
- Организацию главного игрового цикла
- Работу с графикой и отрисовкой объектов
- Обработку пользовательского ввода (клики мышью)
- Управление временем и таймерами
- Детектирование столкновений
- Сохранение и загрузку данных (JSON-файл с рекордом)

## Правила игры

- На экране постоянно появляются и плывут слева направо рыбки двух типов.
- **Обычные рыбы** — красные, меньшего размера. Клики по ним **не учитываются**.
- **Особые рыбы** — зелёные и крупнее. Каждый клик по особой рыбе добавляет **10 очков**.
- Игра длится ровно **60 секунд**. За это время нужно набить максимум очков.
- В верхней части экрана отображаются:
  - **Текущий счёт** (количество набранных очков)
  - **Оставшееся время** (обратный отсчёт с 60 секунд)
  - **Личный рекорд** (лучший результат всех сессий, сохранён в файл)
- Когда время заканчивается:
  - Игра останавливается
  - На экране показывается финальный результат
  - Выводится сообщение о новом рекорде (если он побит)
  - Можно перезапустить игру (клавиша **R**) или выйти (клавиша **ESC**)

## Установка и запуск

### Требования
- **Python 3.6+**
- Библиотека **Pygame**

### Шаг 1: Установите Python (если ещё не установлен)
Скачайте с официального сайта [python.org](https://www.python.org/)

### Шаг 2: Установите Pygame
Откройте терминал или командную строку и выполните:
```bash
pip install pygame
```

### Шаг 3: Скачайте и запустите игру
1. Клонируйте этот репозиторий или скачайте файл `fish_game.py`:
```bash
git clone https://github.com/Mighty1Starling/Fish_game-Python_PyGame.git
cd Fish_game-Python_PyGame
```

2. Запустите игру:
```bash
python fish_game.py
```

3. После первого запуска в папке проекта автоматически появится файл `fish_record.json` с сохранённым рекордом.

## Как работает код

### Основные компоненты

**Класс Fish** — представляет отдельную рыбу:
- `__init__()` — инициализация параметров (позиция, скорость, тип)
- `update()` — обновление координат
- `draw()` — отрисовка рыбы на экран
- `off_screen()` — проверка, вышла ли рыба за границы экрана

**Функция load_record()** — загрузка сохранённого рекорда из `fish_record.json`

**Функция save_record()** — сохранение нового рекорда в JSON-файл

**Функция main()** — главный игровой цикл:
- Обработка событий (клики мыши, нажатия клавиш)
- Спавн новых рыб
- Обновление позиций и удаление вышедших за экран рыб
- Проверка попаданий мышью
- Отрисовка всех объектов и UI
- Управление игровым временем
- Отображение финального экрана

## Решение проблем

**Q: При запуске ошибка "ModuleNotFoundError: No module named 'pygame'"**  
A: Установите Pygame командой: `pip install pygame`

**Q: Файл fish_record.json не создаётся**  
A: Убедитесь, что у приложения есть права на запись в папку. Попробуйте запустить игру от администратора или в другой папке.

**Q: Игра работает рывисто (низкий FPS)**  
A: Это может быть из-за слабого компьютера или других процессов, нагружающих CPU. Закройте другие приложения и попробуйте снова.

**Q: Как изменить количество очков за клик?**  
A: Найдите в коде строку `score += 10` и измените число на нужное значение.
